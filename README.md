## Installation

```
npm install
```

## Getting started

```
npm run android
```

```
npm run ios
```

## Testing

```
npm run test
```

```
npm run e2e:start
```

```
npm run e2e:android
```

```
npm run e2e:ios
```

## Check code

```
npm run lint
```

```
npm run lint:fix
```

```
npm run lint:check
```

```
npm run pretty
```

## Maintenance

```
npm install -g npm-check
```

```
npm-check -u
```
